# 🚀 Featured & WIP Projects
## 🧠 Obsidian RAG Assistant (Active Project)
A local‑first Retrieval‑Augmented Generation engine built directly into Obsidian.
This project is my deep dive into:
- Markdown‑aware chunking
- Incremental indexing pipelines
- Local embeddings (BGE‑M3 via Ollama)
- Vector search (DuckDB or Vectra)
- Hybrid retrieval (semantic + BM25)
- Agentic workflows for vault cleanup, organization, and synthesis
Everything runs locally, with no cloud dependencies.
The goal is simple:
Help people who think in “thought‑dumps” turn chaotic notes into structured, navigable knowledge — without changing how they naturally write.

Repo: [Project Repo](https://github.com/Mizeress/Obsidian-Assistant)

## 🎯 Gun Turret Simulation — Champaign Aviation Museum
At the museum, I’m building a real‑time motion‑tracked simulation using a restored .50 cal turret as the controller.
- C++ + OpenCV for ArUco marker tracking
- Unity for the simulation environment
- Custom native plugin to bridge tracking data into Unity
This project blends computer vision, hardware, and interactive design.

## 📟 Fallout Handheld Device — Embedded Linux Project
A full vertical slice of an embedded Linux system:
- Custom minimal Linux distribution (Yocto/Buildroot)
- Kernel‑level input handling for buttons and analog stick
- Containerized game systems for scalability
- Target: a handheld device capable of running Fallout 1 & 2
This is my foray into embedded systems and low‑level Linux engineering.

Repo: [Project Repo](https://github.com/Mizeress/Fallout-Game-System)
